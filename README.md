# 전북 문화재 관광 플랫폼

## 프로젝트 소개
전북 지역 관광 및 문화재 데이터를 기반으로 관광 정보를 제공하는 플랫폼입니다.

## 기술 스택
- Frontend : Next.js, React, TypeScript
- Backend : Spring Boot, MyBatis
- Database : Oracle
- API : 한국관광콘텐츠랩, Kakao Map API

## 주요 기능
- 관광지 조회
- 문화재 추천
- 지도 기반 위치 표시
- 리뷰 및 북마크
- 관리자 페이지
